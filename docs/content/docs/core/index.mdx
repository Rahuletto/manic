## Architecture Principles

- **Speed Above All**: Every decision, from choosing OXC to using Bun's native server, is driven by performance.
- **Simplicity**: The internal codebase is kept lean and modular.
- **Type Safety**: End-to-end TypeScript ensures a reliable development experience.
