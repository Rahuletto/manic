# `<Link />`

The `<Link>` component provides declarative, client-side navigation with automatic prefetching and View Transitions support.

## Component Signature

```ts
import { Link } from 'manicjs';

type LinkProps = {
  to: string;
  prefetch?: boolean;
  replace?: boolean;
  viewTransitionName?: string;
  className?: string;
  style?: React.CSSProperties;
  children?: React.ReactNode;
};
```

## Examples

### Basic Usage

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export function Nav() {
  return (
    <nav>
      <Link to="/">Home</Link>
      <Link to="/about">About</Link>
      <Link to="/posts">Blog</Link>
    </nav>
  );
}
```

### With Dynamic Path

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export function PostList({ posts }: { posts: Array<{ slug: string; title: string }> }) {
  return (
    <ul>
      {posts.map(post => (
        <li key={post.slug}>
          <Link to={`/posts/${post.slug}`}>{post.title}</Link>
        </li>
      ))}
    </ul>
  );
}
```

### Conditional Replace

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export function BlogLink({ slug, isFeatured }: { slug: string; isFeatured?: boolean }) {
  return (
    <Link
      to={`/blog/${slug}`}
      replace={isFeatured}
      className={isFeatured ? 'font-bold text-lg' : ''}
    >
      {slug}
    </Link>
  );
}
```

### Disable Prefetching

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export function LargeList() {
  // Disable prefetch for large lists to avoid memory pressure
  return (
    <div>
      {Array.from({ length: 1000 }).map((_, i) => (
        <Link key={i} to={`/item/${i}`} prefetch={false}>
          Item {i}
        </Link>
      ))}
    </div>
  );
}
```

### Named view transitions

Optional `viewTransitionName` sets `view-transition-name` on the anchor so you can target this element in CSS for shared-element-style animations:

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export function ProfileLink() {
  return (
    <Link to="/user/1" viewTransitionName="user-card">
      Open profile
    </Link>
  );
}
```

### Disabling view transitions globally

Per-link toggle is not exposed on `<Link>`. Turn transitions off for all navigations via config or runtime:

```tsx twoslash
// @errors: 2322
import { setViewTransitions } from 'manicjs';

setViewTransitions(false);
```

Or set `router.viewTransitions: false` in [`manic.config.ts`](/docs/api/config).

## How It Works

```mermaid
sequenceDiagram
    participant User
    participant Link
    participant Cache
    participant Router
    
    User->>Link: Hover
    Link->>Cache: prefetch route chunk
    Cache-->>Link: Loaded
    User->>Link: Click
    Link->>Router: navigate(to)
    Router->>Cache: Get chunk
    Cache-->>Router: Component
    Router->>Router: Render with View Transition
```

## Features

| Feature | Description |
|---------|-------------|
| **Prefetch** | Loads route JS on hover/focus |
| **Code Splitting** | Each route is a separate chunk |
| **View Transitions** | Smooth animated navigation |
| **Cache** | Loaded chunks in memory |
| **History** | Browser history integration |
