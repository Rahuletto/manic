## Installation

```bash
bun add manicjs
```

All exports are available from the main package:

```ts
// Most exports
import { Link, useRouter, navigate } from 'manicjs';

// Config
import { defineConfig } from 'manicjs/config';

// Environment
import { getEnv } from 'manicjs/env';
```
