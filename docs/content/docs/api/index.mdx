## Installation

The API is fully contained within the `manicjs` package.

```bash
bun add manicjs
```

### Typing
Manic is written in TypeScript and provides first-class type definitions for all exports. We recommend using a modern IDE (like VS Code) to take advantage of the deep autocompletion provided by our type system.
