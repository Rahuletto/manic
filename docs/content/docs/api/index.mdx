## Quick reference

| Export | Preferred import | Description |
| :--- | :--- | :--- |
| `Router`, `Link`, `navigate`, `useRouter`, `useQueryParams` | `manicjs` or `manicjs/router` | SPA routing |
| `preloadRoute`, `RouterContext` | `manicjs/router` only | Advanced routing ([RouterContext](/docs/api/router/router-context)) |
| `defineConfig`, `createPlugin`, `loadConfig`, `getConfig` | `manicjs/config` | Config & plugins ([plugins](/docs/api/config/plugins)) |
| `createManicServer` | `manicjs/server` | Bun server entry ([page](/docs/api/server/create-manic-server)) |
| `createClient` | `manicjs` or `manicjs/client` | Typed RPC ([createClient](/docs/api/client/create-client)) |
| `ViewTransitions`, `setViewTransitions` | `manicjs` or `manicjs/transitions` | View Transitions API |
| `ThemeProvider`, `useTheme`, `ThemeToggle` | `manicjs` or `manicjs/theme` | Theme |
| `initTheme` | `manicjs/theme` only | Early theme apply |
| `getEnv`, `getPublicEnv` | `manicjs/env` | Env access |
| `NotFound`, `ErrorOverlay`, `ServerError` | `manicjs` | Error UI |
| `apiLoaderPlugin`, `fileImporterPlugin` | `manicjs/plugins` | Low-level loaders |

See [**Package exports**](/docs/api/package) for types-only exports (`RouteDef`, `ManicPluginContext`, …).

## Installation

```bash
bun add manicjs
```

```ts
// App code — shortest path
import { Link, useRouter, navigate } from 'manicjs';

// manic.config.ts
import { defineConfig, createPlugin } from 'manicjs/config';

// Advanced routing
import { preloadRoute } from 'manicjs/router';
```
