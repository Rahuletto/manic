# `<NotFound />`

Exported from **`manicjs`** for the router’s **built-in** no-match UI in development-like scenarios. For production UX, add **`app/routes/~404.tsx`** ([Routing](/docs/framework/routing)).

```tsx
import { NotFound } from 'manicjs';
```

## Framework context

The client **`Router`** renders **`NotFound`** when no registry entry matches **`location.pathname`**. Custom **`~404.tsx`** replaces the default experience via **`window.__MANIC_ERROR_PAGES__.notFound`** when configured in **`main.tsx`**.
