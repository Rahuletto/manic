# View Transitions API

Manic wraps the browser [View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transitions_API). When **`router.viewTransitions`** is enabled (default), navigations via **`<Link>`** or **`navigate()`** run inside **`document.startViewTransition`** when available.

## Imports

```ts
import { ViewTransitions, setViewTransitions } from 'manicjs';
// or
import { ViewTransitions, setViewTransitions, navigate } from 'manicjs/transitions';
```

**`navigate`** is re-exported from **`manicjs/transitions`** for convenience.

---

## Configuration

```ts title="manic.config.ts"
import { defineConfig } from 'manicjs/config';

export default defineConfig({
  router: { viewTransitions: false },
});
```
