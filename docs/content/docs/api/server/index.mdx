# Server API (`manicjs/server`)

The server package exposes **`createManicServer`** — everything else (Hono APIs, Markdown negotiation, MCP hints) hangs off that function.

---

## Primary export

<Cards>
  <Card title="createManicServer" description="Bun.serve bootstrap HTML config modes plugins" href="/docs/api/server/create-manic-server" />
</Cards>

```ts
import { createManicServer } from 'manicjs/server';
```

---

## Plugin hooks (`configureServer`)

Plugins extend the Bun router using **`ManicServerPluginContext`**:

<TypeTable
  type={{
    addRoute: {
      type: '(path: string, handler: (req: Request) => Response | Promise<Response>) => void',
      description: 'Attach raw Request handlers.',
    },
    addLinkHeader: {
      type: '(value: string) => void',
      description: 'Append RFC 8288 Link headers on HTML responses.',
    },
    injectHtml: {
      type: '(tags: string) => void',
      description: 'Insert HTML snippets before closing head.',
    },
  }}
/>

Author plugins with **`createPlugin`** ([guide](/docs/api/config/plugins)).
