# `<ThemeToggle />`

Pre-built **`button`** wired to **`useTheme().toggle`**. Supplies **`aria-label`** based on **`isDark`**.

```tsx twoslash
// @errors: 2322
import { ThemeToggle } from 'manicjs';

export function Header() {
  return (
    <header>
      <ThemeToggle className="rounded-lg px-3 py-1 border" />
    </header>
  );
}
```

### Custom content

```tsx twoslash
// @errors: 2322
import { ThemeToggle } from 'manicjs';

<ThemeToggle>
  {(resolved) => (resolved === 'dark' ? 'Light mode' : 'Dark mode')}
</ThemeToggle>;
```
