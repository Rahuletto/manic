# `defineConfig(config)`

Returns the configuration object unchanged while preserving **`ManicConfig`** inference.

```ts title="manic.config.ts"
import { defineConfig } from 'manicjs/config';

export default defineConfig({
  mode: 'fullstack',
  app: { name: 'Docs' },
});
```

---

## Signature

```ts
function defineConfig(config: ManicConfig): ManicConfig;
```

Also available from **`manicjs`** root, but **`manicjs/config`** is the canonical **`manic.config.ts`** import.
