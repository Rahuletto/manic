# Configuration Reference

The `manic.config.ts` file (or `.js`) is the source of truth for your application's build and runtime behavior. Use the `defineConfig` helper to ensure full type safety.

```ts twoslash
import React from 'react';
import { defineConfig } from 'manicjs/config';

export default defineConfig({
  mode: 'fullstack',
  app: {
    name: 'My Manic App',
  },
  server: {
    port: 3000,
  },
});
```

---

## Top-Level Options

| Property | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `mode` | `'fullstack' \| 'frontend'` | `'fullstack'` | `'fullstack'` includes Hono API support. `'frontend'` is SPA only. |
| `app` | `Object` | `{}` | Application-level metadata. |
| `server` | `Object` | `{}` | Development and runtime server settings. |
| `router` | `Object` | `{}` | Client-side navigation settings. |
| `build` | `Object` | `{}` | Production optimization settings. |
| `oxc` | `Object` | `{}` | Transformer and minifier settings. |
| `providers` | `ManicProvider[]` | `[]` | Deployment adapters (Vercel, Cloudflare, etc.). |
| `plugins` | `ManicPlugin[]` | `[]` | Framework extensions. |

---

## `app` Settings

- **`name`**: `string`. The title of your application, injected into the `<title>` tag of the built HTML.

## `server` Settings

- **`port`**: `number`. The port the server binds to. Defaults to `6070`.
- **`hmr`**: `boolean`. Toggle React Fast Refresh in development. Defaults to `true`.

## `router` Settings

- **`viewTransitions`**: `boolean`. Enable the native browser View Transitions API. Defaults to `true`.
- **`preserveScroll`**: `boolean`. If `true`, the router will not reset scroll to top on navigation. Defaults to `false`.
- **`scrollBehavior`**: `'auto' \| 'smooth'`. The CSS scroll behavior for navigation. Defaults to `'auto'`.

## `build` Settings

- **`outdir`**: `string`. The directory for build artifacts. Defaults to `.manic`.
- **`minify`**: `boolean`. Toggle OXC-Minify for production builds. Defaults to `true`.
- **`splitting`**: `boolean`. Enable greedy code-splitting for route chunks. Defaults to `true`.
- **`sourcemap`**: `boolean \| 'inline' \| 'external'`. Source map generation strategy. Defaults to `'inline'`.

## `oxc` Settings

Deeply customize the OXC transform engine.

- **`target`**: `string`. The ECMAScript target. Defaults to `esnext` in dev and `es2022` in prod.
- **`refresh`**: `boolean`. Enable/disable React Fast Refresh transform.
- **`rewriteImportExtensions`**: `boolean`. Automatically converts `.ts` imports to `.js` in the output. Defaults to `true`.

## Sitemap Configuration

Manic can automatically generate a `sitemap.xml` for your application.

```ts
sitemap: {
  hostname: 'https://example.com',
  exclude: ['/admin/**'],
  changefreq: 'daily',
  priority: 0.7
}
```
