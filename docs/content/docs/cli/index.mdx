# CLI Reference

The Manic CLI is the primary way to interact with the framework's build system and development tools.

## Basic Commands

| Command | Purpose |
| :--- | :--- |
| `manic dev` | Start the development server with HMR. |
| `manic build` | Create a production-ready bundle. |
| `manic start` | Serve the production build. |
| `manic lint` | Run `oxlint` diagnostics on your code. |
| `manic fmt` | Format code using OXC. |
| `manic mcp` | Launch the Model Context Protocol server. |

## Global Options

- `--help`: Show usage information.
- `--version`: Show the version of `manicjs`.
- `--cwd <path>`: Set the current working directory.
