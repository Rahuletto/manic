# Framework Guide

Manic is a modern, production-grade React framework built specifically for the **Bun runtime**. It delivers a sub-millisecond development lifecycle by vertically integrating the toolchain using **OXC** for transformation and **Hono** for fullstack routing.

## The Manic Philosophy

Manic is built on three core pillars:
1.  **Vertical Integration**: We eliminate toolchain fragmentation by using Bun and OXC for everything from linting to minification.
2.  **Near-Zero Latency**: Every framework component is optimized to minimize the time between "file save" and "browser update".
3.  **Predictable Fullstack**: API routes and Page routes share the same filesystem conventions and deployment lifecycle.

## Technical Highlights

- **Bundler**: Native Bun.build + custom OXC plugins.
- **Transpiler**: OXC (Ultra-fast TypeScript and JSX transformation).
- **Minifier**: OXC-Minify (Parallel production optimization).
- **Server**: Hono running on the native Bun.serve.
