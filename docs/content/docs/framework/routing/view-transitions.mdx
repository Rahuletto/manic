# View Transitions

<Callout type="info" title="TL;DR">

Manic supports the View Transitions API for smooth, animated **client navigations**. Use [`setViewTransitions`](/docs/api/transitions#setviewtransitionsenabled) or config `router.viewTransitions`, and [`ViewTransitions`](/docs/api/transitions) / `viewTransitionName` on [`Link`](/docs/api/router/link) for shared-element-style CSS.

**Cross-document** transitions are a browser feature between full page loads; Manic focuses on SPA navigations (`history` + React render).

See the [API reference](/docs/api/transitions) for `ViewTransitions` helpers.

</Callout>
## What It Is

View Transitions provide **native browser animations** when navigating between pages:

| Feature | Description |
|---------|-------------|
| **Cross-document** | Animate entire page changes |
| **Container-scoped** | Animate specific components |
| **Shared elements** | Morph elements between pages |
| **Custom animations** | CSS-based keyframe animations |

**Browser Support:** Chrome 111+, Safari 17.4+, Edge 111+

## Quick Start

### Enable View Transitions

```tsx twoslash
// @errors: 2322
// app/routes/index.tsx
import React, { useEffect } from 'react';
import { setViewTransitions, useRouter } from 'manicjs';

export default function Root() {
  useEffect(() => {
    setViewTransitions(true);
  }, []);

  return (
    <div>
      <h1>View Transitions Enabled</h1>
    </div>
  );
}
```

## Global View Transitions

Enable View Transitions globally in your app:

```tsx twoslash
// @errors: 2322
// app/main.tsx
import React from 'react';
import { createRoot } from 'react-dom/client';
import { Router, setViewTransitions } from 'manicjs';

setViewTransitions(true);

declare const document: any;
const root = createRoot(document.getElementById('root')!);
root.render(<Router />);
```

## Custom Transitions

Use CSS `::view-transition-old` and `::view-transition-new` pseudo-elements:

```css
/* app/app.css */
::view-transition-old(root) {
  animation: fadeOut 0.3s ease-out;
}

::view-transition-new(root) {
  animation: fadeIn 0.3s ease-in;
}

@keyframes fadeOut {
  from { opacity: 1; }
  to { opacity: 0; }
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

## Shared Elements

Morph elements between pages using `viewTransitionName`:

```tsx twoslash
// @errors: 2322
// app/routes/index.tsx
import React from 'react';
import { Link } from 'manicjs';

export function Home() {
  return (
    <Link to="/about" viewTransitionName="hero">
      <h1>My Hero</h1>
    </Link>
  );
}

// app/routes/about.tsx
export function About() {
  return (
    <Link to="/" viewTransitionName="hero">
      <h1>My Hero</h1>
    </Link>
  );
}
```

## Disable View Transitions

Disable View Transitions for specific navigations:

```tsx twoslash
// @errors: 2322
import React from 'react';
import { Link } from 'manicjs';

export default function Page() {
  return (
    <Link to="/page" viewTransitionName="fade">
      Go to Page
    </Link>
  );
}
```

Or globally:

```ts twoslash
// @errors: 2322
import { setViewTransitions } from 'manicjs';

setViewTransitions(false);
```
