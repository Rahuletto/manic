# Plugins

<Callout type="info" title="TL;DR">

Plugins extend Manic with features like Tailwind CSS, MDX, SEO, sitemap, and more. Install from npm and configure in `manic.config.ts`.

</Callout>
## What It Is

Manic plugins add functionality without custom code:

| Plugin | Purpose | Package |
|--------|----------|---------|
| **Tailwind CSS** | Styling | `@manicjs/tailwind` |
| **UnoCSS** | Styling | `@manicjs/unocss` |
| **MDX** | Markdown + JSX | `@manicjs/mdx` |
| **SEO** | Meta tags | `@manicjs/seo` |
| **Sitemap** | sitemap.xml | `@manicjs/sitemap` |
| **MCP** | AI tools | `@manicjs/mcp` |
| **API Docs** | OpenAPI UI | `@manicjs/api-docs` |

## Quick Start

### Install Plugin

```bash
bun add @manicjs/tailwind
```

### Configure

```ts
// manic.config.ts
import { defineConfig } from 'manicjs/config';
import { tailwind } from '@manicjs/tailwind';

export default defineConfig({
  plugins: [tailwind()],
});
```

## Plugin Types

### ManicPlugin Interface

```ts
interface ManicPlugin {
  name: string;

  // Build-time
  build?: (ctx: ManicBuildPluginContext) => void | Promise<void>;

  // Dev server
  configureServer?: (ctx: ManicServerPluginContext) => void | Promise<void>;
}

interface ManicPluginContext {
  config: ManicConfig;
  pageRoutes: PageRoute[];
  apiRoutes: ApiRoute[];
  prod: boolean;
  cwd: string;
  dist: string;
}

interface ManicBuildPluginContext extends ManicPluginContext {
  emitClientFile(relativePath: string, content: string | Uint8Array): Promise<void>;
  injectHtml(tags: string): void;
}

interface ManicServerPluginContext extends ManicPluginContext {
  addRoute(path: string, handler: (req: Request) => Response | Promise<Response>): void;
  addLinkHeader(value: string): void;
  injectHtml(tags: string): void;
}
```

### createPlugin Helper

```ts
import { createPlugin } from 'manicjs/config';

const myPlugin = createPlugin({
  name: 'my-plugin',

  staticFiles: [
    {
      path: '/my-file.txt',
      content: 'Hello World',
    },
  ],

  build(ctx) {
    ctx.injectHtml('<meta name="my-plugin" content="true">');
  },
});
```

## Advanced Patterns

### Custom Plugin

```ts
// plugins/my-custom-plugin.ts
import { createPlugin, type ManicPlugin } from 'manicjs/config';

export function myCustomPlugin(): ManicPlugin {
  return createPlugin({
    name: 'my-custom-plugin',

    build(ctx) {
      // Inject custom styles
      ctx.injectHtml(`
        <style>
          .custom-class { color: red; }
        </style>
      `);

      // Emit static file
      ctx.emitClientFile('assets/custom.css', `
        .custom-class { color: blue; }
      `);
    },
  });
}
```

### Plugin Options

```ts
// With options
export default defineConfig({
  plugins: [
    seo({
      title: 'My App',
      description: 'Description',
    }),
  ],
});
```

## Common Issues

### Issue 1: Plugin Not Loading

**Problem:** Plugin not working.

**Checks:**
1. Package is installed
2. Plugin is in `plugins` array
3. Server restarted

### Issue 2: Conflicting Plugins

**Problem:** Two plugins conflicting.

**Solution:** Check order or disable one:

```ts
export default defineConfig({
  plugins: [
    // Only one styling plugin
    // tailwind() or unocss(), not both
  ],
});
```
