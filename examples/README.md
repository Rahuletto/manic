# Examples

Reference applications are **not** committed in this repo. After `./setup.sh`, clones
appear here:

| Directory          | Upstream repo                                                           |
| ------------------ | ----------------------------------------------------------------------- |
| `examples/starter` | [manic-js/example-starter](https://github.com/manic-js/example-starter) |
| `examples/chatbot` | [manic-js/example-chatbot](https://github.com/manic-js/example-chatbot) |

Work on each app in its own git repository; push to `manic-js/example-starter` or
`manic-js/example-chatbot`, not the umbrella.
